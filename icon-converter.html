🎨 StudyTrack Icon Converter

Convert SVG icons to PNG for PWA installation

Choose Your Method

📦 Option 1: Automatic (Node.js)

Fastest method if you have Node.js installed

npm install sharp node convert-icons.js

🌐 Option 2: Online Converter

Use CloudConvert or Convertio to convert SVG to PNG

1. Go to https://convertio.co/svg-png/ 2. Upload: public/icons/icon-192x192.svg 3. Convert & Download as PNG 4. Repeat for icon-512x512.svg 5. Save to: public/icons/

🎯 Option 3: Manual (Browser Canvas)

Use browser developer tools to render and save

1. Open browser DevTools (F12) 2. Go to Console tab 3. Paste the SVG to PNG script below 4. Check your Downloads folder
📋 Steps to Complete PWA Setup
  1. Convert SVG Icons to PNG

    Use one of the methods above to convert:

    icon-192x192.svg → icon-192x192.png icon-512x512.svg → icon-512x512.png
  2. Create Maskable Versions (Optional)

    For better appearance on modern Android devices, create maskable versions with 20% padding inside the icon.

  3. Place Icons in Correct Folder

    Ensure PNG files are in: public/icons/

  4. Test the PWA

    Run npm run dev and open the app on your phone or use Chrome DevTools → Application → Install.

  5. Install to Home Screen

    On iPhone: Share → Add to Home Screen
    On Android: Menu → Install App
